Ruby Methods: Enumerable#cycle

The cycle method

The cycle method offers an easy way to repeat an action on a collection of elements n number of times, or to cycle through a collection infinitely until a break statement interrupts the cycling process.

Simple Definition Creates an infinitely-repeating loop of its items when given a collection.
Version Ruby 1.9
Expects Optional block that acts upon items within the collection in each new iteration of cycle.
Returns #cycle saves elements in an internal array. If no block is given, then an Enumerable::enumerator is returned.

NOTE: Official documentation can be found at ruby-doc.org

How to use the cycle method

  • Passing an argument to cycle will iterate through a collection n number of times, or infinitely if n number of times is not specified.
    • def cycle_example
  letters_in_alphabet = ["a","b","c","d"]
  letters_in_alphabet.cycle(3) {|n| puts n}
end
